# Ejercicio cree un repositorio local
Requiere haber instalado git
## 1. Seleccione el directorio donde quiere crear su repositorio

## 2. Inicialice el repositorio
```
git init
```
## 3. Cree nuevos archivos dentro del directorio seleccionado

## 4. Agregue los archivos al repositorio
```
git add <directorio o archivo>
```
## 5 Haga commit de sus cambios
```
git commit –m "mensaje descriptivo"
```

## 6 Revise el estado sde su repositorio
```
git status
```

## 7 Haga un nuevo branch
```
git branch <nombre_branch>
```

## 8 Cambie al nuevo branch
```
git checkout <nombre_branch>
```

## 9 Agregue nuevos archivos o cambie los actuales

## 10 Agregue los nuevos cambios al branch y haga un commit
```
git add <directorio o archivo>
```

```
git commit –m "mensaje descriptivo"
```

## 11 Vuelva al branch original
```
git checkout main
```

## 12 Combine los cambios del nuevo branch con el branch main
```
git merge <nombre_branch_a_mezclar>​
```
